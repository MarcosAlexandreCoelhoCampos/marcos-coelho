Sobre mim

Olá, eu sou um programador em formação atualmente cursando o 1° semestre em Análise e Desenvolvimento de Sistemas na Cruzeiro do Sul. Meu interesse pela programação começou cedo, aos 14 anos, quando comecei a estudar linguagens como VisualG, Python e PHP por conta própria.

Projetos

WaterSurf

Em apenas uma semana, eu desenvolvi o Watersurf, um projeto desafiador que envolveu a manipulação de DOM com JavaScript e exigiu muito empenho e dedicação. Desde a concepção da ideia até o desenvolvimento completo do código do site, trabalhei arduamente para criar uma plataforma que apresentasse uma escola de surf completa, com diferentes modelos de aula disponíveis, professores e informações de contato.

Durante todo o processo, enfrentei alguns desafios que me exigiram muita criatividade e habilidade técnica, mas com perseverança e determinação, fui capaz de superá-los e finalizar o projeto com sucesso.

CabanaDoCoelho

Com a finalidade de aprimorar minhas habilidades em desenvolvimento web, criei o projeto CabanaDoCoelho durante a minha graduação. O objetivo deste site é apresentar cabanas disponíveis para aluguel e fornecer meios de contato para interessados. Durante o desenvolvimento, enfrente desafios significativos relacionados à dinâmica CSS, como o menu escondido que acabava saindo dos limites da tela e a dificuldade em ajustar as imagens no formato hexagonal. No entanto, através de pesquisas intensivas, principalmente em fontes em inglês, consegui superar esses obstáculos e concluir o projeto com sucesso.

Figma - HTML - CSS

CoelhoGames

O CoelhoGames é um projeto que oferece serviços de assistência técnica para consoles, desenvolvido com o objetivo de aprender Bootstrap. O site foi construído com o uso de HTML, CSS e Bootstrap, e em apenas 5 horas de programação foi possível criar um layout responsivo e intuitivo para os usuários.

Embora tenha sido desenvolvido com o intuito de aprendizado, o CoelhoGames é um exemplo de como o Bootstrap pode ser utilizado para criar um site simples, mas funcional e profissional. Além disso, o projeto demonstra a eficiência do Bootstrap, permitindo a construção de um site de qualidade em pouco tempo.
